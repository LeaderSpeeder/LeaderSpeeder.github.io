Make sure that Wi-Fi Direct is turned on. To check the Wi-Fi Direct on/off status, go to printer control panel or home page (also called the embedded web server or EWS).

Connecting a device that supports Wi-Fi Direct

  1. Turn on wireless on your mobile device or computer, and then activate Wi-fi Direct.
  2. Select the printer's W-Fi Direct name from the list of avalible wireless devices.

Note:

  • You can check the printer's Wi-Fi Direct name on the printer control panel or home page (EWS).
  • If you cannot find the [rinter's Wi-Fi Direct name on the list, go to "Connecting a device that does not support Wi-Fi Direct".
